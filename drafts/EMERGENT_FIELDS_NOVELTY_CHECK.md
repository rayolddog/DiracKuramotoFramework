# Novelty check: "Sakaguchi lag = Haldane flux" — VERDICT: (b) close prior art, chain unpublished

*Date: 2026-07-05. Decisive check for `EMERGENT_FIELDS_PAPER.md`'s novelty anchor.
Question: has "Sakaguchi phase-lag α = Peierls phase/Haldane flux in a Kuramoto
honeycomb lattice, giving C=±1=sign(α) in linearized sync-fluctuation bands with
chiral edge modes" been published? Researched by web agent.*

## Verdict

**(b)** — the specific chain is NOT in print, but three flanks of close prior art
exist. The paper survives only if it positions explicitly against each:

1. **Magnon Haldane via DMI — the sharpest conceptual precedent.**
   Kim, Ochoa, Zarzuela, Tserkovnyak, PRL 117, 227201 (2016); Owerre, J. Phys.
   Cond. Mat. (2016): honeycomb ferromagnet with next-nearest-neighbour
   Dzyaloshinskii–Moriya interaction ⇒ *exactly* the Haldane model for the
   linearized excitations about the aligned state; C = ±1 set by the DMI sign;
   chiral magnon edge modes. Since J·cosΔθ + D·sinΔθ = √(J²+D²)·sin(Δθ+α) with
   tan α = D/J, **the DMI angle IS a Sakaguchi lag in oscillator language** —
   same flux mechanism, published in spin language with Hamiltonian
   (precessional) dynamics. *Omitting this citation kills the paper with any
   magnetism referee.* What the oscillator version adds: first-order dissipative
   (Kuramoto) rather than precessional dynamics — non-Hermitian bands, sync-basin
   context, and platform breadth (JJ/NEMS/spintronic arrays).

2. **Zhu & Zheng, arXiv:2606.24926 (June 2026).** Frustrated Vicsek–Kuramoto
   *continuum* active matter with a Sakaguchi-type lag → non-Hermitian Chern
   numbers C = ±2 of the hydrodynamic dispersion matrix, one-way edge
   circulation. The lag→Chern→chiral-edge *chain* is therefore in print in a
   neighboring setting. Remaining gap our paper owns: the *lattice* setting, the
   *Haldane-flux identification* (lag as Peierls phase on NNN links), C=±1=sign(α)
   for fluctuations about a synchronized state (not pattern-forming instability
   bands), and the honeycomb phase diagram vs. Semenoff mass.

3. **"Topological synchronization" line.** Sone, Ashida, Sagawa, PRR 4, 023211
   (2022); Sone et al., Nat. Phys. 20, 1164 (2024); Wächtler & Platero, PRR 5,
   023021 (2023): topology is *imported by hand* into the coupling matrix
   (QWZ/SSH-type), then synchronization phenomena are studied. Differentiate:
   there, topology is an input; here, it is *generated by the lag* — a native
   sync parameter.

Also adjacent, cite-and-distinguish: Torres-Hugas et al., arXiv:2604.19682
(Sakaguchi lags as gauge field with loop holonomy — the "lag = link phase" idea,
but only as a frustration diagnostic, no band topology); Fruchart et al., Nature
592, 363 (2021) (non-reciprocity → exceptional points, not Chern bands); Nash et
al., PNAS 112, 14495 (2015) (complex hopping from gyroscopy, not delay);
Gavensky et al., PRB 2018 (Haldane in a single multiterminal JJ's synthetic
dimensions, not array phase dynamics).

## Technical caution flagged by the check (must be resolved before submission)

A first-order **real** linearization of K·sin(Δθ+α) about the synchronized state
gives cos α × (graph Laplacian) — Hermitian and topologically trivial. The
referee-proof version of the claim must show exactly where the complex hopping
enters: the inertial (second-order) structure, the amplitude/Stuart–Landau
sector, the NNN geometry, or the non-Hermitian antisymmetric part of the
linearization. Verify what `code/honeycomb_emergence.py` actually linearizes
(does it derive the Bloch Hamiltonian from the oscillator equations, or write
down the Haldane model with φ = α by dictionary?) and close this gap with an
explicit derivation.
